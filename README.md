# Mehedi.EventBus.Kafka
EventBus implementation for Apache Kafka with .NET Core 8.

## Technologies
- .NET Core 8
- C#
- xUnit

## Prerequisites
- net8.0
- Confluent.Kafka (>= 2.3.0)
- Mehedi.EventBus.Abstractions (>= 1.0.0)
- Microsoft.Extensions.Logging.Abstractions (>= 8.0.1)
